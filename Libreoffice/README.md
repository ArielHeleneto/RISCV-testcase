# LibreOffice

## 软件说明

LibreOffice（发音：/ˌliː.brəˈɒ.fɪs/）是由文档基金会开发的自由及开放源代码的办公室套件。LibreOffice套件包含文字处理器、电子表格、演示文稿程序、矢量图形编辑器和图表工具、数据库管理程序及创建和编辑数学公式的应用程序。

## 环境配置

- 操作系统版本： openEuler 22.03 LTS riscv64
- 软件版本： libreOffice 7.3.5.2 30(build:2)
