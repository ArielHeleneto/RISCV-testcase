# Firefox 命令行安装报错失败

## 操作步骤

1. 点击桌面 `terminal` 图标。
2. 输入 `dnf install firefox` 后回车。
3. 按 `y` 键确认。
4. 输入 `dnf install firefox --skip-broken` 后回车。
5. 输入 `dnf install firefox --nobest` 后回车。

## 预期结果

Firefox 安装成功。

## 实际结果

Firefox安装失败。
