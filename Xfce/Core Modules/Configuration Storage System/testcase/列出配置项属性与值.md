# 列出配置项属性与值

## 摘要

- 使用xfconf-query进行命令行式的查看与修改xfconf的配置项

## 操作步骤

1. 输入命令```xfconf-query -c xfce4-panel -l```，查看输出结果
2. 输入命令```xfconf-query -c xfce4-panel -l -v```，查看输出结果
