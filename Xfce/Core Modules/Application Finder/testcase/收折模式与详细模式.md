# 收折模式与详细模式

## 摘要

- 收折模式与详细模式 - 两模式之间的转换

## 操作步骤

1. 打开Application Finder
2. 点击输入框右侧箭头
3. 点击多次，尝试在两者间切换。收折模式下不显示可选应用程序，详细模式下则显示
